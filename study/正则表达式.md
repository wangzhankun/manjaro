# 匹配操作符

```
\	转义字符
.	匹配任意单个字符
[1249a],[^12],[a-k] 	字符序列单字符占位
^ 	行首
$ 	行尾
\<,\>: \<abc	 单词首尾边界
| 	连接操作符，
(,) 	选择操作符
\n 	反向引用
```

# 重复操作符

```
？ 	匹配0到1次
* 	匹配0到多次
+ 	匹配1到多次
{n} 	匹配n次
{n,} 	匹配n到多次
{n,m} 	匹配n到m次
```
